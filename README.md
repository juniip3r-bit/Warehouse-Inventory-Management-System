# Warehouse Inventory Management System

## What it does

This Python application helps manage warehouse inventory using dictionaries. Users can add inventory, process orders, monitor stock levels, and receive low-stock alerts.

## How to run it

1. Open the notebook in Google Colab.
2. Run every cell from top to bottom.
3. Follow the menu prompts.

## Skills Demonstrated

- Dictionaries
- Functions
- While Loops
- Conditional Statements
- User Input
- String Formatting

## What it produces

The application tracks inventory, processes customer orders, and alerts users when stock drops below the minimum threshold.
